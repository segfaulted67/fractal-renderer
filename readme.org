#+TITLE: Fractal Renderer
This is a simple C program that renders fractals using [[https://github.com/raysan5/raylib][raylib]] library.

* Screenshot
  [[./screenshot/mandelbrot.png]]
  [[./screenshot/mandelbar.png]]
* Requirements
  - C compiler(gcc, clang, etc)
  - raylib

* Installation
  #+begin_src sh
  git clone https://github.com/segfaulted67/fractal-renderer
  cd fractal-renderer
  make
  ./fractal
  #+end_src
